# codealpha_task
# 📚 Web Scraping Project using Python

## 📌 Project Overview

This project demonstrates how to extract book information from a website using Python. The scraped data is stored in a CSV file for future analysis.


---

## 🎯 Objective

- Learn the basics of web scraping.
- Extract book titles and prices from a website.
- Store the extracted data in a CSV file.
- Understand HTML structure using BeautifulSoup.

---

## 🛠️ Technologies Used

- Python 3
- Requests
- BeautifulSoup4
- Pandas
- VS Code

---

## 📂 Project Structure

```
webscraping_eda/
│
├── scraper.py
├── books.csv
├── requirements.txt
└── README.md
```

---

## 📦 Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/webscraping_eda.git
```

2. Navigate to the project folder

```bash
cd webscraping_eda
```

3. Install the required libraries

```bash
pip install -r requirements.txt
```

---

## ▶️ How to Run

Run the scraper using:

```bash
python3 scraper.py
```

After execution, a file named **books.csv** will be created containing the scraped data.

---

## 📄 Output

The CSV file contains the following columns:

| Title | Price |
|--------|--------|
| A Light in the Attic | £51.77 |
| Tipping the Velvet | £53.74 |

---

## 📚 Libraries Used

- **requests** – Sends HTTP requests to the website.
- **BeautifulSoup4** – Parses HTML and extracts required information.
- **pandas** – Stores the scraped data and exports it to CSV.

---

## 📖 Learning Outcomes

- Understand how web scraping works.
- Learn to inspect HTML elements.
- Extract data using BeautifulSoup.
- Create datasets in CSV format.
- Work with Python libraries for data collection.
